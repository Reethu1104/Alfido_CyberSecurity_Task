# Alfido CyberSecurity Tasks

This repository contains cybersecurity internship tasks completed as part of practical learning.

## Tasks Included

### Task 1: Web Application Security Basics
- Studied OWASP Top 10 vulnerabilities
- Demonstrated SQL Injection and XSS
- Tools used: WebGoat, Burp Suite

### Task 2: Wireshark Packet Analysis
- Captured and analyzed live network traffic
- Studied HTTP, HTTPS, DNS, TCP, ICMP protocols
- Tool used: Wireshark

### Task 3: Incident Response Simulation
- Simulated a cybersecurity incident
- Followed identification, containment, eradication, and recovery phases
- Tools used: System Logs, Wireshark, Antivirus
